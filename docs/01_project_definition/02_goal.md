# Project Definition - Goal

**Project Type:**    
The project is an application focused project. While recent trends in Natural Language Processing prioritize massive Large Language Models (LLMs), our motivation is to investigate the practical viability of traditional, lightweight Machine Learning algorithms. We aim to determine if these standard techniques—covered in our course—can offer a balance of accuracy and computational efficiency that makes them more accessible for real-world deployment than resource-heavy deep learning alternatives.

**Establish a Baseline:**    
Prove that traditional, lightweight ML algorithms (like Naive Bayes or Logistic Regression) can still achieve high performance for binary text classification.
